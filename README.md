# FastAPI SMS API

A simple SMS Management API built using FastAPI.

## Features

 GET all messages
 GET message by ID
 POST new message
 PUT update message
 PATCH partial update
 DELETE message

## Run

```bash
uvicorn sms:sms --reload
```

## API Documentation

Visit:

http://127.0.0.1:8000/docs

## Technologies

 Python
 FastAPI
 Pydantic
